projects

Take a look at some of my past projects! I mainly work on computer architecture, digital design, and efficient machine learning. In my free time, I also enjoy exploring topics like computer vision, control, and robotics.

SoC Design for Real-Time Data Deduplication and Compression
project page / code

Developed a high-throughput System-on-Chip (SoC) for real-time data deduplication and compression, integrating content-defined chunking (CDC), SIMD-accelerated SHA256 hashing, and an FPGA-based LZW compressor. Attained over 800 Mb/s end-to-end throughput with a 0.65 compression ratio on the Ultra96-V2 platform.

CSRSPMM: Optimized CUDA Kernels for CSR Sparse x Dense MatMul
project page / code

Developed high-performance CUDA kernels for CSR sparse–dense matrix multiplication, leveraging warp-per-row scheduling, shared-memory buffering, and vectorized memory accesses to improve performance on irregular sparsity patterns compared to cuSPARSE and torch.sparse.

Pipelined RISC-V Processor
project page / code

Designed a custom 32-bit RISC-V core in SystemVerilog featuring a five-stage pipelined datapath. The core supports full data forwarding and hazard control logic, includes instruction and data caches, and provides an AXI interface to main memory. The design was validated using the official RISC-V test suite and is deployable on Lattice ECP5 FPGA.

Efficient DNN via Pruning & Sparse Matrix Compression
project page / code

Conducted an empirical analysis of pruning patterns across varying sparsity levels and designed a custom CSR-based linear layer to reduce memory footprint and accelerate inference while preserving model accuracy.

Configurable Logic Block Design, Verification and Optimization
project page

Designed a 16-bit Configurable Logic Block (CLB) in Cadence 45 nm Salicide (1.0 V / 1.8 V, 1P11M) technology. The CLB integrates a 16:1 LUT, SRAM-based memory, SIPO register, and a clock generator. The design was thoroughly verified and optimized to mitigate timing hazards while balancing power and performance.

Auto-driving Engines for F1/10
project page / code

Developed an autonomous driving system for F1/10-scale racing cars, integrating SLAM, motion planning, optimal control, and deep learning–based perception. The system was deployed on the F1TENTH RoboRacer platform.

LC3 Simulator
code

Implemented a C-based simulator for the LC-3 (Little Computer-3) instruction set architecture. The simulator was validated by running benchmark assembly programs, including interactive applications such as 2048, Hangman, and Rogue.

Automatic Solar Tracker
code

Designed a dual-axis automatic solar tracking system to maximize energy absorption of solar panels. The system uses LDR-based feedback and servo motor actuation for panel alignment, implemented on a Tiva TM4C123GH6PM microcontroller.
